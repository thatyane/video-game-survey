# video-game-survey
A video game survey application

## Features
### Mobile
* Choose your favorite game
* Create game (pending)

### Web: https://video-game-survey.netlify.app/
* Listing of chosen games
* Top 8 rated games 
* Dashboard by platforms 
* Dashboard by genre 
* Choose your favorite game (pending)


### Technologies used

* **Java 11**
* **Spring Boot 2**
* **Maven**
* **Junit 5**
* **Mockito**
* **Hamcrest**
* **PostgreSQL 12**
* **Flyway 6**
* **Heroku**
* **Netlify**
